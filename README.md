# 資産管理アプリ

スマートフォンで使える、個人向けのお金の管理アプリです。
給与の手取り計算・クレジットカードの支出管理・家計の予実管理をまとめて行えます。

---

## 画面構成

### カードタブ
JCBとVISAカードの支出を月単位で管理します。

- 固定費・変動費の登録・編集・削除
- 日別支出グラフ（生活費カテゴリ）
- 予算内訳：上限額から固定費・生活費・その他変動費を差し引いた残高を予定/実績で表示
- 生活費（食費・日用品）は別途上限額を設定可能

### 家計タブ
月の収支全体を俯瞰します。

- 2枚合計の使用額サマリー
- 給与・固定費・カード使用額の予定/実績比較
- 生活費カード（週別・今月の進捗バーグラフ）
- カテゴリ別グラフ・集計

### 給与タブ
残業時間を入力するだけで、その月の手取り金額をリアルタイム計算します。

- 職能給・各種手当・社会保険料を自由に編集
- 雇用保険・所得税は実際の明細に合わせた計算式を採用
- 給与履歴（過去明細の閲覧・推移グラフ）

---

## 設定

歯車アイコンから各種設定を変更できます。

| 設定項目 | 内容 |
|---------|------|
| 給与設定 | 基本給・手当・控除項目の編集 |
| カード設定 | 保有カード・締め日・支払い日・上限額 |
| 給与履歴 | 過去明細の確認・推移グラフ |
| データ管理 | データのエクスポート・インポート |

---

## データについて

入力したデータはすべて**お使いのスマートフォン・PCのブラウザ内にのみ保存**されます。
外部サーバーやクラウドへの送信は一切ありません。
ブラウザのデータを削除すると設定も消えますのでご注意ください。

---

## 起動方法

**Node.js** が必要です。

```bash
npm install    # 初回のみ
npm run dev    # 開発サーバー起動
```

起動後、ブラウザで `http://localhost:5173` を開いてください。

```bash
npm run build  # 本番ビルド
```
