# 資産管理アプリ

スマートフォンで使える、個人向けのお金の管理アプリです。
給与明細の手取り計算・クレジットカードの支出管理・銀行口座の残高確認などをまとめて行えます。

---

## できること

### ✅ 給与シミュレーション（実装済み）
残業時間を入力するだけで、その月の**手取り金額をリアルタイムで計算**します。
職能給・各種手当・社会保険料などの金額は、昇給・昇格に合わせて自由に編集できます。

### 🔜 クレジットカード管理（準備中）
毎月かかる固定費（サブスク・光熱費など）と、その月に使った変動費をまとめて管理します。
JCBとVISAそれぞれの締め日・引き落とし日ごとに集計します。

### 🔜 銀行口座・証券口座（準備中）
財布の現金・銀行・証券口座など複数の口座残高をひとまとめに管理します。
給与の受け取りや口座間の振り替えをイベントとして登録すると、残高の推移を確認できます。

### 🔜 過去データ（準備中）
過去の源泉徴収票や月ごとの手取り・残業時間をグラフで振り返ることができます。

## 使い方

### 初めて使うとき

このアプリを動かすには **Node.js** が必要です。
インストール済みの場合は、以下の手順でアプリを起動できます。

```
npm install    ← 必要なファイルをダウンロードする（初回のみ）
npm run dev    ← アプリを起動する
```

起動後、ブラウザで `http://localhost:5173` を開いてください。

### 動作確認済みの端末

- iPhone 13 Pro
- Galaxy Z Fold5

---

## データについて

入力したデータはすべて**お使いのスマートフォン・PCのブラウザ内にのみ保存**されます。
外部のサーバーやクラウドには送信されないので、プライバシーの心配はありません。
ただし、ブラウザのデータを削除すると設定も消えますのでご注意ください。
