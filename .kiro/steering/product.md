# Product Overview

個人の収支・資産をスマートフォンで一元管理するWebアプリ。
サーバー不要・データはブラウザ内のみに保存するため、プライバシーを保ちながら手軽に使える。

## Core Capabilities

- **給与シミュレーション**: 残業時間を入力すると手取り額をリアルタイム計算。固定項目は昇給時に編集可能
- **クレジットカード管理**: 固定費・変動費を登録し、JCB・VISAそれぞれの締め日・引き落とし日で集計
- **口座残高シミュレーション**: 複数の銀行・証券口座残高をイベントベースで管理し、推移を可視化
- **過去データ**: 源泉徴収票・月別給与・残業時間の履歴をグラフで振り返る

## Target Use Cases

- 毎月の手取り見込みを残業時間から素早く確認したい
- クレカの引き落とし額を締め日ベースで把握したい
- 複数口座の残高を俯瞰して資産の全体像を掴みたい

## Value Proposition

- **完全オフライン**: localStorage のみ使用。外部サーバー・APIへの通信なし
- **モバイルファースト**: iPhone / Android を主ターゲットとした UI（max-width 600px）
- **日本語専用**: 日本の給与・税制に特化した計算ロジック
